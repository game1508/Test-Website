Math Notes

Standard Integrals

1.
$\displaystyle \int x^n\,dx = \frac{x^{n+1}}{n+1} + C$
$\displaystyle \int (ax+b)^n\,dx = \frac{(ax+b)^{\,n+1}}{a(n+1)} + C,\; n \neq -1$
2.
$\displaystyle \int \frac{1}{x}\,dx = \ln|x| + C$
$\displaystyle \int \frac{1}{ax+b}\,dx = \frac{\ln|ax-b|}{a} + C$
3.
$\displaystyle \int e^{x}\,dx = e^{x} + C$
$\displaystyle \int e^{ax+b}\,dx = \frac{e^{ax+b}}{a} + C$
$\displaystyle \int a^{x}\,dx = \frac{a^{x}}{\ln a} + C$
$\displaystyle \int a^{px+q}\,dx = \frac{a^{px+q}}{p\,\ln a} + C$
4.
$\displaystyle \int \sin x\,dx = -\cos x + C$
$\displaystyle \int \sin(ax+b)\,dx = -\frac{\cos(ax+b)}{a} + C$
5.
$\displaystyle \int \cos x\,dx = \sin x + C$
$\displaystyle \int \cos(ax+b)\,dx = \frac{\sin(ax+b)}{a} + C$
6.
$\displaystyle \int \sec^2 x\,dx = \tan x + C$
$\displaystyle \int \sec^2(ax+b)\,dx = \frac{\tan(ax+b)}{a} + C$
7.
$\displaystyle \int cosec^2 x\,dx = -\cot x + C$
$\displaystyle \int cosec^2(ax+b)\,dx = -\frac{\cot(ax+b)}{a} + C$
8.
$\displaystyle \int \sec x \cdot \tan x \, dx = \sec x + C$
$\displaystyle \int \sec(ax+b)\tan(ax+b)\,dx = \frac{\sec(ax+b)}{a} + C$
9.
$\displaystyle \int cosec x \cdot \cot x \, dx = -cosec x + C$
$\displaystyle \int cosec(ax+b)\cot(ax+b)\,dx = -\frac{cosec(ax+b)}{a} + C$
10.
$\displaystyle \int \frac{1}{\sqrt{1-x^2}}\,dx = \sin^{-1}x + C = -\cos^{-1}x + C$
11.
$\displaystyle \int \frac{1}{1+x^2}\,dx = \tan^{-1}x + C = \cot^{-1}x + C$
12.
$\displaystyle \int \frac{1}{|x|\sqrt{x^2-1}}\,dx = \sec^{-1}x + C = -cosec^{-1}x + C$
13.
$\displaystyle \int \frac{1}{\sqrt{a^2-x^2}}\,dx = \sin^{-1}\left(\frac{x}{a}\right) + C$
14.
$\displaystyle \int \frac{1}{x^2+a^2}\,dx = \frac{1}{a}\tan^{-1}\left(\frac{x}{a}\right) + C$
15.
$\displaystyle \int \frac{1}{|x|\sqrt{x^2-a^2}}\,dx = \frac{1}{a}\sec^{-1}\left(\frac{x}{a}\right) + C$
16.
$\displaystyle \int \frac{1}{x^2}\,dx = -\frac{1}{x} + C$
17.
$\displaystyle \int \frac{1}{\sqrt{x}}\,dx = 2\sqrt{x} + C$
18.
$\displaystyle \int k\,dx = kx + C$
19.
$\displaystyle \int \tan x \, dx = \ln|\sec x| + C$
20.
$\displaystyle \int \cot x \, dx = \ln|\sin x| + C$
21.
$\displaystyle \int \sec x \, dx = \ln|\sec x + \tan x| + C$
22.
$\displaystyle \int cosec x \, dx = \ln|cosec x - \cot x| + C$
